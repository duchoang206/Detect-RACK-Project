Bước 1: Cài đặt các thư viện phụ thuộc (Dependencies)

Vì dự án của bạn sử dụng OpenCV, Modbus và IXWebSocket, khách hàng cần đảm bảo máy của họ đã cài các thư viện này. Ví dụ trên Ubuntu/Linux:
Bash

sudo apt-get update
sudo apt-get install build-essential cmake libopencv-dev libmodbus-dev

Bước 2: Tự biên dịch ra file chạy mới

Khách hàng mở Terminal tại thư mục dự án và chạy chuỗi lệnh sau để tự tạo ra file chạy phù hợp 100% với cấu hình máy của họ:
Bash

mkdir build
cd build
cmake ..
make -j$(nproc)

Sau khi chạy xong lệnh make, một file thực thi (ví dụ: DetectRackProject) mới tinh sẽ tự động xuất hiện trong thư mục build/ của khách hàng và họ có thể chạy bình thường bằng lệnh:
Bash

./DetectRackProject
